# CERIF-DataModel

The **C**ommon **E**uropean **R**esearch **I**nformation **F**ormat (= CERIF) is a flexible data model 
for representing research information (i.e., information about research). 
It was developed and is being maintained by [euroCRIS](http://www.eurocris.org/).

This branch is for publishing the model documentation only.
Please switch to the [master](https://github.com/EuroCRIS/CERIF-DataModel/blob/master/) or [develop](https://github.com/EuroCRIS/CERIF-DataModel/blob/develop/) branch for more information.
